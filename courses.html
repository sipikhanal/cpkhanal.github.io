Database Systems-(CST2355Z)

Database systems automate data processing and security of IT systems. Students gain experience using tools like Oracle and MySQL, along with hands-on practice with advanced SQL, data storage/retrieval, and data warehousing.


Operating System Fundamentals (Gnu/Linux)-(CST8102Z)

An exploration of operating system fundamentals and their interaction with hardware and software. Students will examine OS structures, process management, storage management, and more, all in the context of the GNU/Linux operating system.


Object Oriented Programming (Java)-(CST8284Z)

Object-Oriented Programming (OOP) concepts are crucial for programmers. In this course, students will use Java to explore OOP principles like encapsulation, inheritance, abstraction, and polymorphism, as well as data structures, algorithms, and basic GUI programming.


Web Programming-(CST8285Z)

Develop web programming, website design, and implementation skills with JavaScript, HTML5, and PHP. Explore interactive web-based solutions while constructing and maintaining your own websites.


Technical Communication for Engineering Technologies-(ENL2019Z)

Strengthen technical communication skills crucial for a tech-focused workplace. Develop independent and collaborative critical thinking, research, writing, visual communication, and presentation skills.


Globalization And Sustainability-(PSI0003Z)

This course examines the complex impacts of globalization on the economy, environment, politics, and culture. The course will explore questions of sustainability, cultural exchange (and conflict), and the everyday implications of living in an increasingly interconnected world.
